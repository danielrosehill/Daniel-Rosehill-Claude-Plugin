# Daniel-Rosehill

Personal-use Claude Code plugin for Daniel Rosehill. Released publicly via [Daniel's marketplace](https://github.com/danielrosehill/Claude-Code-Plugins) for convenience — not intended as a general-purpose plugin, but you're welcome to use it.

## Skills

- **view-github-repos** — Open Daniel's public GitHub repository list (latest, by stars, or A–Z) in the browser.

## Installation

```bash
claude plugins marketplace add danielrosehill/Claude-Code-Plugins
claude plugins install Daniel-Rosehill@danielrosehill
```
